Function Reference

move(int amount) — Moves the turtle the specified number of pixels in the direction it's facing.

poop() — Causes the turtle to place an image of poop behind it.

run(int amount) — Moves the turtle the specified number of pixels in the direction it's facing at twice the rate of move.

turn(int amount) — Turns the turtle counter-clockwise the specified number of degrees. Using a negative value causes the turtle to turn clockwise.

turnAndMove(int turnAmount, int moveAmount) — Turns and moves the turtle at the same time.

int getWorldWidth() and int getWorldHeight() — Retrieves the width and height of the world the turtle can move around in.

int getTurtleWidth() and int getTurtleHeight() — Retrieves the width and height of the turtle.

int getX() and int getY() — Retrieves the horizontal and vertical positions of the turtle in the world.

int getAngle() — Retrieves the angle the turtle is facing.

void alert(String message) — Display a message box

Math Functions

int abs(int value) — Returns the absolute value (greater than 0) of the value parameter.

int remainder(int dividend, int divisor) — Returns the remainder of dividend / divisor
